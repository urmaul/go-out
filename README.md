go-out
======

External links hiding and redirecting script.

How to install
--------------

1. Copy script dir to your site dir.
2. Create "config.php" file with config and links list.

Config file sample
------------------

```php
<?php

return array(
	'links' => array(
		'link1' => 'http://link1.com/',
		'link2' => 'http://link2.com/',
	),
);
```
